# Java SpringBoot+Vue酒店后台管理系统 java258

## 项目概述

本项目是基于Java SpringBoot和Vue的前后端分离的酒店后台管理系统。系统致力于提供完善的酒店管理功能，包括客房管理、入住与预定管理，以及留言板等。

## 技术栈

- **数据库：** MySQL
- **开发工具：** IDEA、VsCode、Navicat
- **数据连接池：** Druid
- **Web容器：** Apache Tomcat
- **版本控制工具：** Git
- **项目构建工具：** Maven
- **后端技术：** SpringBoot、MyBatis-Plus
- **前端框架：** Vue2、Vue Router、Axios、Element-UI、file-saver

## 功能模块

### 酒店管理员

- **登录注销**
- **客房管理**
  - 查询客房
  - 入住管理
  - 预定管理（支持数据导出Excel）

### 系统管理员

- 员工模块（添加、删除、修改、查询员工信息）

## 系统角色

- 酒店管理员
- 系统管理员

## 运行环境

- 数据库：MySQL
- 服务器：Apache Tomcat
- IDE：IDEA（推荐）

## 部署步骤

1. 使用Navicat或其他数据库管理工具，在MySQL中创建数据库，并导入项目提供的SQL文件。
2. 使用IDEA导入项目，配置好相应的Tomcat服务器。
3. 修改项目中`src/main/resources/jdbc.properties`文件，配置正确的数据库连接信息。

## 核心亮点

- 前后端分离设计，便于管理和维护。
- 使用SpringBoot和Vue，提高开发效率。
- 系统功能完整，满足酒店日常运营需求。
- 采用Druid数据连接池，优化数据库性能。
- 通过Git进行版本控制，确保代码质量。
- 支持Excel文件导出，方便数据统计与分析。
